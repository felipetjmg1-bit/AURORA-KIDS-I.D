# Perguntas Frequentes

## Como faço para instalar o AURORA-KIDS-I.D?

Veja a seção 'Início Rápido' no README.md para instruções de instalação.

## Quais são os requisitos de sistema?

- Python 3.8+
- PostgreSQL 12+
- 2GB de RAM mínimo
- 10GB de espaço em disco

## Como faço para relatar um bug?

Veja a seção 'Como Relatar Bugs' em CONTRIBUTING.md.

## Posso usar isto em produção?

Este projeto está em desenvolvimento ativo. Recomendamos aguardar a versão 1.0 para uso em produção.
