# Contribuindo para AURORA-KIDS-I.D

Obrigado por considerar contribuir para o AURORA-KIDS-I.D! Este documento fornece diretrizes e instruções para contribuir.

## Código de Conduta

Todos os contribuidores devem aderir ao nosso Código de Conduta. Esperamos comportamento profissional e respeitoso em todas as interações.

## Como Relatar Bugs

Bugs são rastreados como problemas do GitHub. Ao criar um relatório de bug, inclua:

- Um título claro e descritivo
- Uma descrição detalhada do comportamento observado
- Passos específicos para reproduzir o problema
- Exemplos específicos para demonstrar os passos
- Descrição do comportamento observado e o que você esperava ver
- Capturas de tela e/ou logs de erro, se aplicável

## Sugestões de Melhorias

As sugestões de melhorias são rastreadas como problemas do GitHub. Ao criar uma sugestão, inclua:

- Um título claro e descritivo
- Uma descrição detalhada da funcionalidade sugerida
- Exemplos de como a funcionalidade seria usada
- Casos de uso específicos

## Pull Requests

- Siga o guia de estilo Python (PEP 8)
- Inclua testes apropriados
- Atualize a documentação conforme necessário
- Termine todos os arquivos com uma nova linha

## Guia de Estilo

- Use Python 3.8+
- Siga PEP 8 para estilo de código
- Use type hints onde apropriado
- Escreva docstrings em português
