# Guia de Segurança

## Relatando Vulnerabilidades

Se você descobrir uma vulnerabilidade de segurança, **não** crie um problema público. Em vez disso, envie um email para security@impulsodigital.com.br.

## Práticas de Segurança

Este projeto segue as melhores práticas de segurança:

- Criptografia AES-256 para dados em repouso
- TLS 1.3 para dados em trânsito
- Validação de entrada em todos os endpoints
- Rate limiting para prevenir abuso
- Logs de auditoria completos

## Conformidade

- LGPD (Lei Geral de Proteção de Dados)
- GDPR (General Data Protection Regulation)
- ISO 27001 (em processo de certificação)
